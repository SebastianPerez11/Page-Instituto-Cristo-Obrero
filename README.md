# Page-Instituto-Cristo-Obrero

Página Web del Instituto Cristo Obrero, fundado en 1936, de gestión privada, mixto, de jornada simple y con orientación en valores de la religión Católica, disponiendo de infraestructura religiosa propia.

## Descripción general
Se desarrollo una página web dónde se encuentra la información más importante, cómo datos de contacto, se presenta los servicios y su infraestructura. Se desarrollo bajo la metología lineal de desarrollo.

### Tecnologías

- HTML5
- CSS
- Javascript
- Vercel
- Librerías
- Google Fonts
